# phyr97 marketplace

Personal Claude Code plugin marketplace.

## Plugins

| Plugin | Version | Description |
|---|---|---|
| [deep-research](https://github.com/phyr97/deep-research) | v0.4.0 | Deep research across web, codebase, and knowledge domains with auto-scaling |
| [plugin-insights](https://github.com/phyr97/plugin-insights) | v0.2.0 | Analyzes how well Claude Code plugins perform by evaluating JSONL session transcripts |

## Install

Add the marketplace, then install plugins:

```
/plugin marketplace add phyr97/phyr97-marketplace
/plugin install deep-research@phyr97
/plugin install plugin-insights@phyr97
```

# phyr97 marketplace

Personal Claude Code plugin marketplace.

## Plugins

| Plugin | Description |
|---|---|
| [deep-research](https://github.com/phyr97/deep-research) | Deep research across web, codebase, and knowledge domains with auto-scaling |
| [plugin-insights](https://github.com/phyr97/plugin-insights) | Analyzes how well Claude Code plugins perform by evaluating JSONL session transcripts |

## Install

```bash
claude marketplace add phyr97/phyr97-marketplace
```
